# Superstore Sales Data Analysis (Power BI Dashboard)

## 📊 Project Overview
This project features a comprehensive sales analysis dashboard created using the *Superstore Sales Dataset* from Kaggle. The goal of this analysis is to provide actionable business insights into profitability, regional performance, and product category trends to support data-driven decision-making.

## 🛠️ Tools & Skills Used
* *Microsoft Excel:* Used for initial data exploration and preprocessing.
* *Power BI Desktop:* Designed the end-to-end interactive dashboard and visualizations.
* *DAX (Data Analysis Expressions):* Developed advanced measures for MTD, QTD, and YTD time-intelligence analysis.
* *Power Query:* Executed the ETL process, including data cleaning and transformation.

## 🚀 Key Features
* *Interactive Slicers:* Dynamic year-based filtering (2016-2019) for focused data drill-downs.
* *KPI Monitoring:* Real-time visibility of Total Sales ($1.10M), Total Profit ($132.52K), and Profit Margin (12.05%).
* *Regional Performance:* Graphical breakdown of profit and sales across West, East, Central, and South regions.
* *Product Insights:* Deep dive into top-selling products and sub-category sales performance.
* *Time Intelligence:* Integrated complex DAX calculations to track business growth over time.
